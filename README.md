# Box Chain Arena V4.5

Box Chain Arena is a childhood paper strategy game reborn as a modern online duel game.

This prototype includes:
- Guided learning demo
- Offline gameplay
- Offline 2-player mode
- Firebase-based online rooms
- Registered player portal
- Unique nickname system
- Online players list
- Challenge invite system
- Custom duel arenas
- Virtual Entry Gold system
- Gold, stars, diamonds, and shop rewards

## Important Notice

Entry Gold and all rewards are virtual in-game rewards only. They have no real money value, cannot be withdrawn, and are not linked to any cash payment or gambling system.
